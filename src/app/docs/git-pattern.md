# Git commit patterns:


### Main

```
🎉 :tada: init:
🎨 :art: style: 
🔧 :wrench: fix: 
📚 :books: doc: 
🔨 :hammer: refactor: 
✨ :sparkles: feat:
🐛 :bug: bugfix: 
🐎 :racehorse: peform:
⏪ :rewind: revert: 
✔️ :heavy_check_mark: test: 
🔖 :bookmark: tag: 
⚡ :zap: update:
🔒 :lock: security: 
⬆️ :arrow_up: upgrade:
⬇️ :arrow_down: downgrade:
➖ :heavy_minus_sign:
➕ :heavy_plus_sign:
🔀 :twisted_rightwards_arrows: merge:
🚧 :construction: wip:
🚚 :truck: move:
```

### Adding scope

```
🔨 :hammer: refactor(home): discribe the activity
```

### Adding specification

```
🐛 :bug: bugfix #123: discribe the bug
```

### Adding Status

```
🎨 :art: style[init]: discribe the activity
🎨 :art: style[wip]: discribe the activity
🎨 :art: style[complete]: discribe the activity
```
